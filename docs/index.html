    source

    Source Query library for crystal

    Installation

    1. Add the dependency to your shard.yml:

      dependencies:
  source:
    github: Rorkh/source

    2. Run shards install

    Usage

    require "source"

query = Source::Query.new "37.230.162.62", 27015
info = query.info

puts info.players

query.close
